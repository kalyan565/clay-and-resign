## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
```

### Deployment

This project is configured for GitHub Pages deployment. The site will be available at:
`https://kalyan565.github.io/clay-and-resign/`
